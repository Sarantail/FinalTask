1. Просим ввести пользователя текстовый массив с элементами, записанными через пробел.
2. Разбиваем элементы массива.
3. Проверяем размер элементов на соответствие условию (меньше 3 символов).
4. Если элемент удовлетворяет этому условию, записываем его во второй массив.
5. Повторяем пункты 3-4, пока массив не подойдёт к концу.
6. Выводим второй массив.
